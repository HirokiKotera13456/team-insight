# Team Axis Insight

**軸スコア制 × チーム相性可視化 SaaS**

Team Axis Insight は、
**仕事上の行動・判断の傾向を数値で可視化**することで、

* なぜチームの議論が噛み合わないのか
* なぜ決断が遅くなるのか
* チームにどんな偏りがあるのか

を、**人事・マネージャーでも直感的に理解できる形**で示す
B2B 向けの自己学習プロジェクトです。

---

## 1. このプロジェクトで解決したいこと

> 「このチーム、なぜうまく回らないのか？」

この疑問に対して、

* 性格論
* 相性論
* 感覚的な評価

ではなく、

👉 **数字と傾向** で説明できる状態を目指します。

※ 心理学的な正しさを証明するものではありません。
※ あくまで「チーム運営のヒント」を得るためのツールです。

---

## 2. 考え方の特徴（重要）

### ❌ やらないこと

* 16タイプ性格診断
* 性格ラベル付け
* 商標・専門理論への依存

### ✅ やること

* 行動や判断の **傾向を数値化**
* チーム全体の **平均とばらつき** を見る
* 話し合いや意思決定が詰まりやすい理由を説明する

---

## 3. 軸スコアとは？

各メンバーは、以下の **4つの観点** について
**0〜100 のスコア** を持ちます。

> 50 が「どちらでもない」状態です。

| 観点      | 左寄り       | 右寄り      | 仕事で分かること   |
| ------- | --------- | -------- | ---------- |
| 行動エネルギー | よく考えてから動く | まず動く     | 決断の速さ      |
| 判断基準    | 人の気持ち重視   | 論理・事実重視  | 意見が衝突しやすい点 |
| 進め方     | 柔軟に変えたい   | 計画通り進めたい | 進行方法のズレ    |
| 視点      | 現実・具体     | 抽象・将来    | 話が噛み合わない理由 |

👉 **性格を決めつけるものではありません**
👉 「仕事の進め方の傾向」を見るための指標です

---

## 4. どんな入力をするのか

* 簡単な質問に答えるだけ
* 各観点につき数問（合計12問）
* スライダー形式で直感的に回答

結果として、以下のような数値が保存されます。

```json
{
  "行動エネルギー": 68,
  "判断基準": 42,
  "進め方": 81,
  "視点": 55
}
```

---

## 5. チームでは何が分かるのか

### ① チームの平均

* チーム全体がどんな傾向か
* 行動派が多いのか、慎重派が多いのか

### ② チームのばらつき（重要）

* ばらつきが大きい → 意見が割れやすい
* ばらつきが小さい → 偏りやすい

#### 例

* 「進め方」のばらつきが大きい

  * 計画派と柔軟派で揉めやすい

* 「判断基準」が論理寄りで揃っている

  * 外部から冷たく見られる可能性

---

## 6. 相性スコアについて

* メンバー同士の考え方の距離を数値化
* 0〜100 で表示

重要なのは、

👉 **なぜその点数なのかを説明できること**

「感情面と進め方の考えが大きく違うため、衝突しやすい」
といった形で、理由を文章で示します。

---

## 7. 画面イメージ（出力）

### 個人向け

* 自分の傾向レーダー
* 強み・注意点のコメント

### チーム向け

* チーム全体の傾向
* 意見が割れやすいポイント
* 改善のヒント

---

## 8. システム構成（簡単に）

* Web 画面：Next.js
* ログイン：Firebase Authentication
* データ保存：Cloud Firestore
* 集計処理：Cloud Functions

※ サーバー管理は不要
※ 集計は裏側で自動処理

| メンバー | 閲覧・自己入力   |
