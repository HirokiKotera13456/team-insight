# Work Insight

**自分自身の職場での立ち位置を理解するための可視化ツール**

Work Insight は、
**仕事上の行動・判断の傾向を数値で可視化**することで、

* 自分自身の仕事の進め方の特徴を客観的に把握する
* 職場での自分の立ち位置や役割を理解する
* 自分の強みや傾向を数値で認識する

を、**エンジニアでなくても直感的に理解できる形**で示す
自己理解のための自己学習プロジェクトです。

※ 本ツールは性格診断ではありません。
※ 「仕事の進め方の傾向」を整理し、自分自身の職場での立ち位置を理解するための補助ツールです。

---

## 1. このプロジェクトで解決したいこと

> 「自分自身の職場での立ち位置を客観的に理解したい」

仕事の現場では、

* 自分がどのような働き方をしているのか
* 職場で自分がどのような役割を担っているのか
* 自分の仕事の進め方の特徴は何か

といったことを、感覚ではなく数値で把握したいというニーズがあります。

本プロジェクトでは、これらを

👉 **感覚ではなく、数値と傾向** で自分自身を理解できる状態

を目指します。

---

## 3. 軸スコアとは？（個人理解）

各ユーザーは、以下の **4つの観点** について
**0〜100 のスコア** を持ちます。

> 50 は「どちらでもない」状態です。

| 観点      | 左寄り       | 右寄り      | 仕事で分かること   |
| ------- | --------- | -------- | ---------- |
| 行動エネルギー | よく考えてから動く | まず動く     | 決断の速さ      |
| 判断基準    | 人の気持ち重視   | 論理・事実重視  | 意見のぶつかりやすさ |
| 進め方     | 柔軟に変えたい   | 計画通り進めたい | 仕事の進行スタイル  |
| 視点      | 現実・具体     | 抽象・将来    | 話の噛み合い方    |

👉 **良し悪しはありません**
👉 「仕事の癖」を把握するための指標です

---

## 4. 何を入力するのか

* 簡単な質問に答えるだけ
* 各観点につき数問（合計12問）
* スライダー形式で直感的に回答

入力結果は、以下のような形で保存されます。

```json
{
  "行動エネルギー": 68,
  "判断基準": 42,
  "進め方": 81,
  "視点": 55
}
```

## 5. まず個人で分かること

* 自分は慎重派か行動派か
* 論理重視か、感情重視か
* 計画型か、柔軟型か

これらを **レーダーチャート** で可視化します。

## 6. 自分自身の職場での立ち位置を理解する

自分のスコアを分析することで、

* 自分がどのような働き方の傾向を持っているか
* 職場で自分がどのような役割を担いやすいか
* 自分の強みや特徴が数値で明確になる

これらを **文章で説明** することで、

👉 **自分自身の職場での立ち位置を客観的に理解** できます。

---

## 7. 継続的な自己理解のための活用

定期的に診断を行うことで、

* 自分の傾向が時間とともにどう変化するか
* 職場環境や役割の変化が自分の傾向にどう影響するか
* 自分の成長や変化を数値で追跡できる

が見えるようになります。

※ 自分自身の理解を深めるための継続的なツールとして活用できます。
