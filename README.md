# Team Insight

**仕事の進め方を数値で理解する個人向け可視化ツール**
（チーム分析にも拡張可能）

Team Insight は、
**仕事上の行動・判断の傾向を数値で可視化**することで、

* なぜ自分と他人の考え方が噛み合わないのか
* なぜ意思決定でストレスを感じるのか
* なぜ同じ場面で人によって判断が違うのか

を、**エンジニアでなくても直感的に理解できる形**で示す
自己理解・相互理解のための自己学習プロジェクトです。

※ 本ツールは性格診断ではありません。
※ 「仕事の進め方の傾向」を整理するための補助ツールです。

---

## 1. このプロジェクトで解決したいこと

> 「なぜ、この人と話すと話が噛み合わないのか？」

仕事の現場では、

* 同じ説明をしているのに伝わらない
* 決断のスピードが合わない
* 進め方で毎回すれ違う

といったことが頻繁に起こります。

本プロジェクトでは、これらを

👉 **相性や感覚ではなく、数値と傾向** で説明できる状態

を目指します。

---

## 2. このツールの考え方（重要）

### ❌ やらないこと

* 16タイプ性格診断
* 性格ラベルによる分類
* 心理学理論・商標への依存

### ✅ やること

* 行動や判断の **傾向を数値で表す**
* 自分と他人の違いを可視化する
* 違いが衝突につながる理由を説明する

---

## 3. 軸スコアとは？（個人理解）

各ユーザーは、以下の **4つの観点** について
**0〜100 のスコア** を持ちます。

> 50 は「どちらでもない」状態です。

| 観点      | 左寄り       | 右寄り      | 仕事で分かること   |
| ------- | --------- | -------- | ---------- |
| 行動エネルギー | よく考えてから動く | まず動く     | 決断の速さ      |
| 判断基準    | 人の気持ち重視   | 論理・事実重視  | 意見のぶつかりやすさ |
| 進め方     | 柔軟に変えたい   | 計画通り進めたい | 仕事の進行スタイル  |
| 視点      | 現実・具体     | 抽象・将来    | 話の噛み合い方    |

👉 **良し悪しはありません**
👉 「仕事の癖」を把握するための指標です

---

## 4. 何を入力するのか

* 簡単な質問に答えるだけ
* 各観点につき数問（合計12問）
* スライダー形式で直感的に回答

入力結果は、以下のような形で保存されます。

```json
{
  "行動エネルギー": 68,
  "判断基準": 42,
  "進め方": 81,
  "視点": 55
}
```

---

## 5. まず個人で分かること

* 自分は慎重派か行動派か
* 論理重視か、感情重視か
* 計画型か、柔軟型か

これらを **レーダーチャート** で可視化します。

---

## 6. 他人と比べると何が分かるか

* どの観点が大きく違うか
* どこで衝突しやすいか
* なぜ話が平行線になりやすいか

相性スコア（0〜100）とともに、

👉 **なぜその結果になるのか** を文章で説明します。

---

## 7. チームに広げると見えること（発展）

個人の結果を集めることで、

* チーム全体の傾向
* 意見が割れやすいポイント
* 偏りによるリスク

が見えるようになります。

※ チーム分析は個人分析の延長線上にあります。
